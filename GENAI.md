# Transparency

This disclosure exists in the interest of transparency for contributors, users, and downstream forks.

The goal is to openly acknowledge the use of modern development tooling within the project.

Generative AI may be used to assist with:

- code scaffolding
- debugging assistance
- refactoring suggestions
- workflow automation
- boilerplate generation
- technical research assistance

All generated content included in the project is reviewed and tested before inclusion in the repository.

Generative AI systems are treated as development tools rather than autonomous contributors.

---

# Human Oversight

Human contributors remain responsible for:

- gameplay design
- code review
- testing
- maintainability

No code or generated output is merged automatically without human review.

---

# Philosophy

This project views generative AI as a tool that can help accelerate experimentation, reduce repetitive work, and improve accessibility for independent developers.

The use of AI assistance does not remove the need for:

- testing
- maintainability
- contributor accountability

---

# Licensing and Attribution

All contributions to this repository remain subject to the repository license regardless of whether generative AI tools were involved in their creation.

Contributors are expected to avoid knowingly submitting material that violates third-party copyrights, licenses, or terms of use.
